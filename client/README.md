CineStream is a Netflix-inspired subscription-based streaming platform built as a full-stack web application. It allows users to browse, search, and stream movies and TV shows with a personalized, dark-themed UI designed for an immersive entertainment experience.
✅ User Authentication (Register / Login / Logout)
✅ JWT Token-based Security
✅ Homepage with Dynamic Content Rows
✅ Movie/TV Show Detail Pages
✅ Video Player with Controls
✅ Search Functionality
✅ Category-based Browsing
✅ User Profile Management
✅ Responsive Design (Mobile + Desktop)
